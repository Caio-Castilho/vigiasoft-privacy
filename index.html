Política de Privacidade — VigiaSoft Mobile

Última atualização:

Esta Política de Privacidade descreve como o aplicativo VigiaSoft Mobile (“App”) coleta, utiliza e protege dados. O App é destinado ao controle operacional de segurança/portaria, com recursos como autenticação, cadastro de visitantes e registro de entrada/saída.

1) Dados que podemos coletar

  • Dados de autenticação: usuário e credenciais para login (processadas pela API do sistema).
  • Dados de uso operacional: registros de entrada/saída, visitantes, ocorrências e informações relacionadas.
  • Localização (quando habilitada): latitude/longitude para validar presença e registrar eventos quando necessário.
  • Fotos (opcional): imagem vinculada ao cadastro do visitante, quando você seleciona/fornece uma foto.
  • Dados técnicos: informações de diagnóstico e logs básicos para suporte e correções.

2) Como usamos os dados

  • Permitir login e manter a sessão do usuário.
  • Executar funcionalidades do sistema (cadastro, consulta e registros).
  • Sincronizar dados com o servidor quando houver internet (modo “offline-first”).
  • Melhorar estabilidade e investigar falhas/ANRs quando aplicável.

3) Armazenamento local e modo offline

O App pode armazenar dados localmente no dispositivo (ex.: banco local/SQLite) para funcionamento offline. Quando a conexão estiver disponível, o App tenta sincronizar as alterações com o servidor.

4) Compartilhamento de dados

Os dados operacionais são enviados para a API do VigiaSoft para processamento e armazenamento conforme o uso do sistema. Não vendemos dados pessoais. Podemos compartilhar dados quando exigido por lei ou para proteger direitos e segurança do serviço.

5) Permissões do dispositivo

  • Localização: para recursos de verificação/registro, quando habilitado.
  • Câmera/arquivos: para selecionar/enviar foto de visitante, quando você optar.
  • Internet: para comunicação com a API e sincronização.

6) Segurança

Adotamos medidas razoáveis para proteger os dados. Ainda assim, nenhum método de transmissão ou armazenamento é 100% seguro.

7) Retenção e exclusão

Dados podem permanecer no dispositivo enquanto necessários ao funcionamento offline e no servidor conforme as regras operacionais do sistema. Para solicitações de exclusão/retificação, use o contato abaixo.

8) Crianças

O App não é destinado a crianças menores de 13 anos e não tem como objetivo coletar dados de crianças.
